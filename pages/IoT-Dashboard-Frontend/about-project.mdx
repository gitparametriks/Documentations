# About The Project

The **IoT Dashboard Frontend** is a feature-rich, responsive web application built by **Parametriks** to provide an intuitive interface for monitoring and managing IoT devices. The dashboard offers real-time data visualization and control functionalities to optimize and monitor IoT operations efficiently.

## Key Features

- Real-time monitoring of IoT devices.
- Data visualization for various metrics including claims, risks, and sensor data.
- AWS Cognito integration for authentication and user management.
- Comprehensive analytics for clients and insurers.
